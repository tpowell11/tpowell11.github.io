Other Related Projects

  Along with building the rockets, I have also written software having to do with rocketry. Currently, my main rocketry-related programming project is cpprocket, a command-line rocket design and simulation system. As of 8/21/21, it is unfinished and buggy, and also sports some questionable to unredeemable code quality issues. I work on it often, however, both other obligations and lack of skill slow development. The first release version will be once the first simulator has been implemented. It will be a simple Euler's Method simulator, however more complex methods like Runge Kutta will be implemented in later versions. For more information see the project README.
